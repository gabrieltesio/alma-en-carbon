# Alma en Carbón

Artista visual especializado en retratos y obras personalizadas en grafito y carbón.  
Identidad · memoria · proceso creativo

## Sobre el proyecto
Alma en Carbón es un proyecto artístico centrado en la figura humana, la memoria y el proceso manual como forma de pensamiento.

## Técnica
- Grafito
- Carbón
- Papel
